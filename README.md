# Pahana Edu Billing System

A simple Java application to manage customer accounts and billing for Pahana Edu Bookshop in Colombo City.

## Features

- ✅ **User Authentication with secure login**

- 📝 **Customer Account Management** : Add, update, view or delete customers
  
- 📦 **Item Management** : Add, update, or delete item details

- 💵 **Calculate and Print Bill**

- ℹ️ **Help Section**

- ❌ **Exit System**

## Tech

- Language: **Java**
- Data Storage: **Text Files**, Java Collections
